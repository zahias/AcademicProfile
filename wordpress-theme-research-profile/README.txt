== Installation ==

1. Download the theme files
2. In your WordPress admin, go to Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Choose the research-profile-theme.zip file
5. Click "Install Now"
6. Activate the theme

== Quick Start Guide ==

1. **Activate the Theme**: Go to Appearance > Themes and activate "Research Profile Theme"

2. **Create Your First Researcher Profile**:
   - Go to Researchers > Add New in your WordPress admin
   - Fill in the basic information (name, title, bio)
   - Add the OpenAlex ID (find it at openalex.org)
   - Set current affiliation details
   - Upload a profile photo as featured image
   - Publish the profile

3. **Sync OpenAlex Data**:
   - Click the "Sync OpenAlex Data" button in the meta box
   - Wait for the synchronization to complete
   - Publications and analytics will be automatically populated

4. **Customize Display**:
   - Go to Appearance > Customize
   - Select a featured researcher for the homepage
   - Configure theme options as needed

== Frequently Asked Questions ==

= What is OpenAlex? =
OpenAlex is a comprehensive catalog of scholarly papers, authors, institutions, and more. It provides free, open access to academic research data that this theme uses to automatically populate researcher profiles with publication data, citations, and analytics.

= How do I find my OpenAlex ID? =
Visit openalex.org and search for your name. Your OpenAlex ID will be in the format "A1234567890" (starting with 'A' followed by numbers).

= Can I use this theme without OpenAlex integration? =
Yes, you can create researcher profiles with just manual information entry. However, the advanced analytics and publication listings require OpenAlex data.

= Is the theme mobile responsive? =
Absolutely! The theme is built with a mobile-first approach and works beautifully on all device sizes.

= Can I customize the colors and styling? =
Yes, the theme uses CSS custom properties that can be easily customized. You can modify the colors in the style.css file or through custom CSS.

= How many researchers can I add? =
There's no limit to the number of researcher profiles you can create. The theme includes efficient pagination and listing features to handle any number of profiles.
